# ai-coding-agent

## 项目概述

这是一个基于 AI 的代码生成与辅助工具，旨在通过集成多种 AI 模型和工具，提升开发效率。

## 主要功能

- **AI 代码生成**：利用 AI 模型（如 OpenAI、Ollama）自动生成或优化代码。
- **开发工具集成**：支持文件读写、系统时间获取、文件列表查看等常用开发操作。
- **类型安全**：使用 TypeScript 提供类型检查，确保代码的健壮性。
- **自动化测试**：内置单元测试框架，支持对工具功能进行验证。

## 依赖与配置

### 核心依赖
- `ai`: 6.0.99
- `openai`: 6.25.0
- `ollama-ai-provider-v2`: 3.3.1
- `zod`: 4.3.6

### 开发依赖
- `tsx`: 4.21.0
- `typescript`: 5.9.3
- `@biomejs/biome`: 2.4.4
- `committier`: 0.2.3
- `lefthook`: 2.1.1

### 工具配置
- 使用 `biome` 进行代码格式化与检查。
- 通过 `lefthook` 实现 CI/CD 自动化流程。
- 支持 `pnpm` 作为包管理器。

## 项目结构

- `src/`: 源码目录
  - `main.ts`: 主入口文件。
  - `agent.ts`: AI 代理逻辑。
  - `models.ts`: 定义 AI 模型与工具。
  - `tools/`: 提供各种工具函数（如读写文件、获取系统时间等）。
- `evals/`: 单元测试目录，包含对工具功能的测试用例。
- `tsconfig.json`: TypeScript 配置文件。
- `package.json`: 项目配置文件。

## 使用方式

1. 安装依赖：`pnpm install`
2. 启动开发服务器：`pnpm dev`
3. 运行测试：`pnpm test`
4. 执行脚本：`pnpm start` 启动应用。
